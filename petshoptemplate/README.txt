Foi feito em HTML e CSS e a funcionalidade do calendário utilizou JavaScript. As diversas funcionalidades estão implementadas nos diversos arquivos inclusos no .zip

No arquivo areaadmin.html, estão as funcionalidades da área do administrador; nela o administrador pode cadastrar outros administradores, clientes e produtos.

O arquivo calendário.html, contém as datas para o agendamento de serviços. O cliente pode selecionar data que deseja e aparecerá os horários disponíveis para escolher.

No arquivo cadastro.html o cliente pode cadastrar seus pets. É neste arquivo que o cliente fará login, bem como administrador do sistema.

O arquivo cliente.html está a opção de agendamento de horário.

O arquivo contato.html tem informações sobre o PetShop, como endereço e horário de funcionamento, e telefone e e-mail para contato, e as redes sociais onde ele está presente.

O arquivo dia.html faz parte da operação de agendamento de serviço.

O arquivo guiapet.html trás curiosidades sobre os pets e cuidados que devemos ter com eles. Lá o usuário poderá ver todas as postagens de notícias realizadas.

O arquivo index.html é a página principal do site.

O arquivo login.html é a página de autenticação de usuários.

No arquivo petprodutos.html, são listados os produtos comercializados na loja eletrônica e física do PetShop.

O diretório CSS contém o arquivo style.css que contém a formatação das páginas .html

No diretório images, está as imagens utilizadas no trabalho.
